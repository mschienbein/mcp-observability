🧪 MCP UI Integration Test Page

This page tests the MCP UI rendering functionality in LibreChat

📋 Testing Instructions

  1. Open LibreChat in your browser (usually http://localhost:3090)
  2. Start a new conversation with a model that has MCP tools enabled
  3. Try these commands in the chat:
    • Show me the user registration form
    • Display the system dashboard
    • Show me a data table with pagination
    • Create a bar chart visualization
    • Show me the settings panel
  4. Check the browser console for debug logs (press F12)
  5. Look for these console messages:
    • ✅ MCP UI Resource detected! - Resource was found in tool output
    • ToolCallInfo - Storing MCP UI resource: - Resource is being stored
    • MCP UI Component Detected! - HTML block contains UI component

Expected MCP UI Resource Format

Tool Output Structure

{ "type": "resource", "resource": { "uri": "ui://form/user-registration", "name": "User Registration Form", "mimeType": "text/html", "text": "<div>...HTML content...</div>" } }
